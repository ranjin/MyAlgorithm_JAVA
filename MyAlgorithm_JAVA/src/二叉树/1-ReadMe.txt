二叉树的思维模式：
1. 
【遍历】的思维模式
是否可以通过遍历一遍二叉树得到答案？
如果可以，用一个traverse函数配合外部变量来实现。

2.
【递归】的思维模式
是否可以定义一个递归函数，通过子问题的答案推导出原问题的答案?
如果可以，写出这个递归函数的定义。

3. 
所谓前序位置，就是刚进入一个节点的时候，
后序位置，就是即将离开一个节点的时候。

4. 
中序位置主要用在BST场景中，BST的中序遍历结果就是一个有序数组

5. 
前序位置的代码是自顶向下的。
后序位置的代码是自底向上的。

6. 
后序位置的特点：只有后序位置才能通过返回值获取子树的信息。
一旦发现题目和子树有关，大概率需要在后序位置写代码。
